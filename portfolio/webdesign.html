Drawing with code

For our second project, I was tasked with recreating a picture of a porkroll box and a typing book, using HTML and CSS. I started out by plotting the larger shapes and then worked in more and more detail. As a bonus, I added a hover action over the fingers so they appear to be typing on the keyboard.

television

To learn how to create a website that houses webpages, I designed a television that changes channels when the users click buttons on the remote. I modeled the TV after 1950s television sets for a challenge because modern flatscreens are very simple and easy to draw with HTML and CSS. For the channels, I embedded YouTube videos of clips from my favorite shows and movies: Lovecraft Country, Kuch Kuch Hota Hai, HunterxHunter, and Martin.

A part of the assignment was to utilize a grid in the design, so I used it to create the remote, which traditionally uses a grid to display the numbers.

ipod

To practice using radio inputs and button animation, I coded an iPod that changes songs when the users click on the next button on the dial. I chose my favorite songs and artists: Otis Redding, Beyonce, Aaliyah, Anita Baker, and Nancy Ajram. I modeled after an iPod rather than an iPhone because I found the gradient and texture of the iPod more challenging to draw with HTML and CSS. For the button, I utilized radio inputs to change the iPod song display and hid any indicator it was used for a seamless design.

performative design

I collaborated with a fellow student to create a webpage that documented or lead a performance. Our idea was to have the user follow an object that would eventually lead to performing the act DJ spinning a record. We decided on the object being a brown circular entity named Cocoa that would float around the page, instructing the user to follow it with their mouse.

The performance is coded to automatically start when the webpage visted, and the entire action takes 2 minutes to complete.

lava lamp

After being assigned to create a webpage that uses HTML, CSS, and JAVA Script, I built an interactive lava lamp. The lava lamp changes to random colors every time the page reloads and the bubbles move throughout the lamp and grow when the cursor hovers over the glass. I used Java Script to randomize the color change of the lava lamp and stop the colors from continuously changing on a loop while the page was loaded.

A finalized version of the lava lamp is also present on my landing page.
